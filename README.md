# Zombie Survival

Survive as many waves of zombies as you can... [Play here!](https://jemgunay.co.uk/zombiesurvival)

## Usage

```bash
# run locally
npm install
npm run serve
# build dist for release
npm run build
# lint
npm run lint
```
